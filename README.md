# Cows_And_Bulls
A simple game developed using python with a tkinter User Interface. 
## Setup Instructions 
1. Clone the repository <br>
`git clone https://github.com/tanvi0904/Cows_And_Bulls.git`
2. Change the current working directory <br>
`cd Cows_And_Bulls`
3. Execution <br>
`python pro_gui_6.py`
